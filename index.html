Registrás cada cobro, subís el comprobante del depósito y OBRAERP prepara todos los datos de la factura automáticamente. Con un clic se copian al portapapeles y se abre el portal de ARCA para que solo tengas que pegar y confirmar.
Instalar como app en celular / tablet
OBRAERP funciona 100% offline — se abre desde un ícono en tu pantalla de inicio, sin internet, igual que una app nativa.
Android
  1. Abrí el archivo OBRAERP.html en Chrome
  2. Tocá el menú ⋮ (tres puntos, arriba a la derecha)
  3. Tocá "Añadir a pantalla de inicio"
  4. Confirmá con "Añadir"
  5. ¡Listo! Aparece el ícono en tu pantalla 🎉
💡 Para pasar el archivo al teléfono: WhatsApp (mandatelo a vos mismo), USB, Google Drive o Bluetooth.
iPhone / iPad (iOS)
  1. Abrí el archivo en Safari (no Chrome)
  2. Tocá el botón Compartir ⬆️ (abajo en el centro)
  3. Deslizá y tocá "Agregar a pantalla de inicio"
  4. Cambiá el nombre si querés → "Agregar"
  5. ¡Listo! Ícono en tu pantalla de inicio 🎉
💡 Para pasar el archivo: AirDrop desde la PC, o guardalo en Archivos → iCloud Drive y abrilo desde ahí con Safari.
🌐 Versión online gratuita (opcional) — sincronizá entre dispositivos
Si querés acceder desde cualquier dispositivo sin pasar archivos, podés publicar OBRAERP en GitHub Pages (gratis, privado/público).
Con esto: URL fija tipo tuusuario.github.io/obraerp, funciona offline como PWA real, instalable desde cualquier navegador.
GitHub Pages — paso a paso (10 minutos, gratis):
  1. Creá cuenta en github.com (gratis)
  2. Nuevo repositorio → nombre: obraerp → Público
  3. Subí el archivo OBRAERP.html renombrado como index.html
  4. En Settings → Pages → Source: main branch → Save
  5. Tu URL: tuusuario.github.io/obraerp
  6. En el celular: abrís la URL → Chrome/Safari → Añadir a pantalla de inicio
  7. Funciona offline automáticamente (los datos siguen en tu dispositivo)
